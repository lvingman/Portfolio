Homefix

Gestor de inventario de herramientas y artículos de construcción para el hogar que permite administrar y controlar el flujo de productos.
Facilita un seguimiento preciso de las existencias, optimiza los niveles de inventario, previene escasez o exceso de productos, agilizando las operaciones y mejorando las tomas de decisiones.

Tecnologias aplicadas:

  • .Net Core 6 / C#
  • Entity Framework
  • PostgreSQL
  • Json With Token (JWT)

Meritos en proyecto:

  • Mediador entre las ramas de backend y frontend
    • Diseñador de la version original de la base de datos
  • Escritor de la documentacion de los endpoints

Techoil

CRUD para administracion de operaciones y usuarios de una empresa petrolifera.
Dividido en dos secciones: desarrollo de la API y consumo de esta

Tecnologias aplicadas:

  • .Net Core 6 / C#
  • Entity Framework
  • SQL Server
  • MVC
  • Repository Pattern
  • Json With Token (JWT)
  • SALT cryptography

Meritos en proyecto:

  • Desarrollador de ambas API y Aplicacion Web
    • Aprendizaje de encriptacion de contraseñas, patron repositorio, y Entity Framework "Code-First"
  • Implementacion de CORS, JWT

Kirifan

Sitio web para un equipo de competicion de automoviles electricos.
Permite la funcionalidad de administrar usuarios, los cuales estan divididos en Community Managers e ingenieros del equipo, organizar sponsors, y tambien un bulletin board para actualizar novedades.

Tecnologias aplicadas:

  • .Net Core 6 / C#
  • Entity Framework
  • SQL Server
  • Simple Mail Transfer Protocol (SMTP)

Meritos en proyecto:

  • Diseñador de la base de datos
  • Implementacion de registro de usuarios
  • Desarrollo de un sistema de recuperacion de contraseñas vias SMTP
  • Aprendizaje del patron de arquitectura de software MVC y el ORM Entity Framework.
